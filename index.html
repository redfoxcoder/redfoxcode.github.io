HTML and CSS Cheatsheet

Created by Estella Gu

HTML

Here are the common HTML tags and their uses:

  • <h1> : biggest header size
  • <h6> : smallest header size
  • <p> : common paragraph
  • <ol> : ordered list
  • <ul> : unordered list
  • <li> : list items inside
  • <div> : grouping elements

CSS

Here is the structure of a common CSS selector:

[name of selected (ex. h1, .class)] {
any tags (see below);
}

Here are the common CSS tags and their uses:

  • font-size : size(px)
  • color: color(named fonts, RBG, HEX, HSL, and alpha)
  • font-family: font (Times New Roman...)
  • font-weight: (ex. Bold, Light...)
